# /creative-design — Kreatives Design erstellen

Entwirf ein visuell distinktives UI-Design von Grund auf. Lies zuerst den vollständigen Skill und folge dessen Prozess.

## Ablauf

1. Lies `skills/creative-design.md` und folge dem beschriebenen Prozess vollständig
2. Starte mit Design Thinking: Purpose, Tone, Differentiation
3. Wende die Anti-AI-Slop Rules konsequent an
4. Falls `$ARGUMENTS` leer ist, frage den User was designt werden soll

## Kontext

Dieser Command wird typischerweise verwendet:
- Für neue Produkte, Landing Pages, Marketing-Seiten
- Wenn kein bestehendes Design System vorhanden ist
- Wenn visuell distinctive Ergebnisse gefragt sind (nicht generisch)
